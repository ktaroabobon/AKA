# Product Overview

妹のぬいぐるみ「紅（あか）」を LINE Bot 化し、家族と日常的に雑談できる相手を提供する個人プロジェクト。LINE Webhook を受け取り、必要に応じて Gemini ベースの会話バックエンドへルーティングして応答を返す。

## Core Capabilities

- LINE Webhook 受信とメンション判定（家族グループ / 複数人トーク / 1:1 DM の 3 系統）
- 完全一致 / 挨拶 / 自己紹介などの **テンプレ応答** で即応性を担保
- それ以外の発話は **Gemini (`@google/genai`)** ベースの AI バックエンドに転送して会話応答
- **会話履歴** を Firestore に session 単位 (`group:{id}` / `room:{id}` / `user:{id}`) で保持し、直近 20 ターン AND 2h 以内をマルチターン文脈として渡す。24h TTL で自動消滅
- **PII / 不適切語マスキング** を入力 / 出力 / 履歴保存の 3 ポイントで適用し、外部 LLM・履歴のいずれにも原文を残さない
- 上流障害時の **ランダム応答フォールバック** で「無反応」を避ける

## Target Use Cases

- 妹と家族が LINE のグループ / DM で「あか」と日常会話する
- 「おはよう」「ただいま」のような繰り返しの挨拶に温度のある反応を返す
- 個人向けの非業務用途。法人レベルの可用性 / コンプライアンス要件は持たない

## Value Proposition

- ぬいぐるみのキャラクターが LINE 上で「ずっとそこにいる」体験
- LLM 応答とテンプレ応答を組み合わせて、即応性と自然さを両立
- 個人プロジェクトとして **過剰な抽象化を避けたシンプルな構成** を維持
